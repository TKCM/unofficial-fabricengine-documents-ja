Value Editor¶

Value Editor は Canvas 上のノードもしくはメイングラフの値を変更するための入力ウィジェットです。

ノード上のすべての編集可能なポートがテープル上に表示されます。入力に外部からの接続を持つポートは表示されません。

注釈

サブグラフを編集する際、ポートのコンテキストメニュー Edit Port から値のUIの数字幅やコンボボックスの選択肢を編集できます。
