キーボードショートカット¶

Canvas¶

1

選択ノードのポートはすべて非表示

2

選択ノードの接続があるポートのみ表示

3

選択ノードの全ポートを表示

Tab

新規ノード作成検索ダイアログ表示

CTRL+A

全ノードを選択

CTRL+Z
Undo.
CTRL+Y
Redo.
CTRL+X

選択ノードを切り取り

CTRL+C

選択ノードをコピー

CTRL+V

ペースト

CTRL+Tab

ノードエクスプローラ、バリューエディタを表示/非表示

CTRL+R

選択ノードより上流にある全ノードを自動再配置

Alt + left/middle mouse button or Space + left mouse button

移動

Alt + right mouse button

拡縮

Mouse Wheel

拡縮

A

全てのノードをうつす

F

選択ノードをうつす

F2

ノードプロパティダイアログ

ダブルクリック

選択ノードをインスペクト

Shift ダブルクリック

選択ノードを編集

Backspace/Del

選択ノード削除

Fabric Standalone¶

CTRL+N

新規グラフの作成

CTRL+O

既存グラフの読み込み

CTRL+S

グラフの保存

Alt+F4

Fabric スタンドアロンの終了

Q

マニュピレーションのトグル

CTRL+G

3Dビューポート上のグリッドの表示

CTRL+0

Canvas のズームを100％にリセット

CTRL+4

Canvas のグラフを表示/非表示

CTRL+5

エクスプローラを表示/非表示

CTRL+6

バリューエディタの表示/非表示

CTRL+7

undo/redoヒストリの 表示/非表示

CTRL+8

ログメッセージの表示/非表示

CTRL+9

タイムラインの表示/非表示

Alt + left mouse button

カメラ回転

Alt + right mouse button
