Introduction and Background¶

An RTVal is a data container used to pass data values between the Fabric Engine and client applications. This guide explains the details of how RTVals function, as well as how to use them from your C/C++ or Python applications.

This document assumes that you are familiar with many elementary concepts in Fabric Engine. If you haven’t already, we recommend that you read the Core Overview.
