Canvas¶

Canvas is the new programming interface to the Fabric Core that can be accessed both visually and through scripting. Canvas is not covered in detail here; instead, please refer to Canvas ユーザガイド, Canvas Programmer Guide and Canvas Core API for more information about using Canvas at various levels.
