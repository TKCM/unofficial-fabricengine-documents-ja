Documentation System Prerequisites¶

Fabric Engine version 2.0.0
Copyright 2010-2015 Fabric Software Inc. All rights reserved.

Once you have installed sphinx (http://sphinx-doc.org/) you can test it in python like this:

import sphinx
print sphinx.__version__

You need to have the Python subfolder of the Fabric Engine deployment on you python path (for example $FABRIC_DIR/Python/2.7) to gain access to the Fabric Core python client as well as the Documentation System’s sphinx directives.

You can test both of these modules like this:

import FabricEngine.Core as fe
from FabricEngine.Sphinx.ASTWrapper import *

Once all of these snippets work without errors you are ready to generate docs.
