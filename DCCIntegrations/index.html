Read Me First!¶

With 2.0 of Fabric Engine the main integration in all DCCs is Canvas. Prior to 2.0 the main integration was Splice, you can find the previous documentation in these pages as well.

Canvas¶

Fabric is integrated in a series of DCCs (Digital Content Creation Tools).

If you are not familiar with Canvas in general please see:

  • Fabric Engine 入門
  • Canvas ユーザガイド
  • Canvas プログラマーガイド

  • Fabric for Maya
  • Fabric for Softimage
