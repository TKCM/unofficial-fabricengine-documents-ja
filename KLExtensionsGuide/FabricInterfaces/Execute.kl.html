Execute (struct)¶

バージョン 2.0.0 で追加.

The Execute data type is used to represent execution within Canvas. It’s purely used to provide some data which can be passed through the graph without any usage. The merge function only exist to result in Canvas presets through KL2DFG. That way users can hook up multiple execution branches in sequence.
