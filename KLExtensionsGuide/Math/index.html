Math Extension¶

Fabric Engine version 2.0.0
Copyright 2010-2015 Fabric Software Inc. All rights reserved.

The Math extension provides a 3d math library used throughout Fabric Engine.

The Fabric Engine Core exposes the basic low level Math constructs, such as Scalar, Integer and Boolean values, along with basic numerical methods. The documentation of the Core Math services can be found in the Functions and Other Global Declarations document.

Trigonometric functions can be found here: Trigonometric Functions.

Rounding functions can be found here: non_transcendental_functions_functions-functions.

The Math.kl file provides a list of constants and helper functions.
