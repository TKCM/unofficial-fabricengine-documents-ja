Types¶

Vec2_d (struct)¶

The Vec2_d type represents a vector with 2 components. The Vec2_d is also used as the row type for the Mat22.

参考

Vec3, Vec4

/*
** Example: Vec2_d
*/

Vec2_d v(0.0, 0.0);
