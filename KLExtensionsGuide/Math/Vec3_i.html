Vec3_i (struct)¶

The Vec3_i type represents a vector with 3 components. The Vec3_i is also used as the row type for the Mat33.

注釈

Vec3_i utilizes the KL Vector Functions

参考

Vec2, Vec4, Xfo

/*
** Example: Vec3_i
*/

Vec3_i v(0.0, 0.0, 0.0);
