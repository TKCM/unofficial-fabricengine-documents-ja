Types¶

Vec4_i (struct)¶

The Vec4_i type represents a vector with 4 components. The Vec4_i is also used as the row type for the Mat44.

注釈

Vec4_i utilizes the KL Vector Functions

参考

Vec2, Vec3_i

/*
** Example: Vec4_i
*/

Vec4_i v(0.0, 0.0, 0.0, 0.0);
