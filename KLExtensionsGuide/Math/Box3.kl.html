Box3 (struct)¶

The Box3 represents a 3 dimensional bounding box. It uses Vec3 types for each of the 2 elements.

参考

Box2

/*
** Example: Construct Box3
*/

// create a box3
Vec3 min(0.0, 0.0, 0.0);
Vec3 max(1.0, 1.0, 1.0);
Box3 box(min, max);
