Mat22 (struct)¶

The Mat22 represents a 2 by 2 matrix. It uses Vec2 types for each of the 2 rows.

/*
** Example: Construct Mat22
*/

// create a matrix 2x2
Vec2 v0(1.0, 0.0);
Vec2 v1(0.0, 1.0);
Mat22 mat(v0, v1);
