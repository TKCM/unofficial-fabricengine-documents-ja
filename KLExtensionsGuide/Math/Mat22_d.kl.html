Types¶

Mat22_d (struct)¶

The Mat22_d represents a 2 by 2 matrix. It uses Vec2_d types for each of the 2 rows.

/*
** Example: Construct Mat22_d
*/

// create a matrix 2x2
Vec2_d v0(1.0, 0.0);
Vec2_d v1(0.0, 1.0);
Mat22_d mat(v0, v1);
