Vec3 (struct)¶

The Vec3 type represents a vector with 3 components. The Vec3 is also used as the row type for the Mat33.

注釈

Vec3 utilizes the KL Vector Functions

参考

Vec2, Vec4, Xfo

/*
** Example: Vec3
*/

Vec3 v(0.0, 0.0, 0.0);
