Vec4 (struct)¶

The Vec4 type represents a vector with 4 components. The Vec4 is also used as the row type for the Mat44.

注釈

Vec4 utilizes the KL Vector Functions

参考

Vec2, Vec3

/*
** Example: Vec4
*/

Vec4 v(0.0, 0.0, 0.0, 0.0);
