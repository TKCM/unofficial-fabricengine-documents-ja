Box2 (struct)¶

The Box2 represents a 2 dimensional bounding box. It uses Vec2 types for each of the 2 elements.

参考

Box3

/*
** Example: Construct Box2
*/

// create a box2
Vec2 min(0.0, 0.0);
Vec2 max(1.0, 1.0);
Box2 box(min, max);
