Vec3_d (struct)¶

The Vec3_d type represents a vector with 3 components. The Vec3_d is also used as the row type for the Mat33.

注釈

Vec3_d utilizes the KL Vector Functions

参考

Vec2, Vec4, Xfo

/*
** Example: Vec3_d
*/

Vec3_d v(0.0, 0.0, 0.0);
