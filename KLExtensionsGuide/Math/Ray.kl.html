Ray (struct)¶

The Ray type represents linear 3D ray. This is expressed by a start and direction Vec3

/*
** Example: Ray
*/

Ray ray(Vec3(0.0, 0.0, 0.0), Vec3(0.1, -1.0, 0.4));
