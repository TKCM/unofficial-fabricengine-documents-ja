BulletConcaveShape (interface)¶

The BulletConcaveShape class provides an interface for non-moving (static) concave shapes. It has been implemented by the BulletStaticPlaneShape, BulletBvhTriangleMeshShape and BulletHeightfieldTerrainShape.
