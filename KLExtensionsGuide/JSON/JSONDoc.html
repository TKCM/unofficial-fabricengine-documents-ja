JSONDoc (object)¶

バージョン 1.13.0 で追加.

The main JSON document. It provides functionality for encoding and parsing of JSON strings.

参考

JSONDoc.write, JSONDoc.parse

JSONDoc.merge! ( in JSONDoc other )

Merges a second provided JSONDoc into this one. Please see the JSON Extension for a code example.

Boolean JSONDoc.parse! ( in String json )

parses a provided JSON String into the JSONDoc. The valid member will be set after that and can be checked by the user. Please see the JSON Extension for a code example.

String JSONDoc.write! ( in Integer indentation )

Encodes the documents content into a String using a provided indentation setting. Please see the JSON Extension for a code example.

String JSONDoc.write! ()

Encodes the documents content into a String using an indentation of 0. Please see the JSON Extension for a code example.
