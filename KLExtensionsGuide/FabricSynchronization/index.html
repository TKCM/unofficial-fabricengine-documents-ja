FabricSynchronization Extension¶

Fabric Engine version 2.0.0
Copyright 2010-2015 Fabric Software Inc. All rights reserved.

The FabricSynchronization extension contains functions or objects that can help with synchronization.

The SimpleLock object implements a simple mutex.

The FewObjectsRecyclingAllocator object implements a simple per-thread object allocator.
