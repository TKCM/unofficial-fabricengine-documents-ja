Vec2Attribute (object)¶

Specialized GeometryAttribute containing one Vec2 per element. Values can be accessed with the Vec2Attribute.values array member, but its usage might be restricted by the owner Geometry.

Vec2 Vec2Attribute.getLinearCombination? ( in LocalL16UInt32Array sourceIndices, in LocalL16ScalarArray sourceWeights )

Returns a linear combination of source values.

sourceIndices The array of source indices
sourceWeights The array of source weights, which need to correspond to each sourceIndices. Normally the sum of the weights should be 1.0

Size Vec2Attribute.getScalarValueSize? ()

If all attribute values can be expressed as a constant Scalar array, returns the constant number of elements, 0 otherwise.
