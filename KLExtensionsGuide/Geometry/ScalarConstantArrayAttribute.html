ScalarConstantArrayAttribute (object)¶

Specialized GeometryAttribute containing a constant sized arrays of Scalar, packed as a single memory buffer. The ScalarConstantArrayAttribute.setArraySize method sets a constant array size for all values. Values can be accessed with the ScalarConstantArrayAttribute.values array member, but its usage might be restricted by the owner Geometry.
