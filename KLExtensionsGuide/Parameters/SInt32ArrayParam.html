SInt32ArrayParam (object)¶

バージョン 1.12.0 で追加.

The SInt32ArrayParam (object) implements the Param (interface) interface for SInt32 array values.





SInt32ArrayParam param = :ref:`SInt32ArrayParam <sint32arrayparam>`(‘counts’);
param.resize(3);
param.setValue(0, 3);
param.setValue(1, 8);
param.setValue(2, 12);
report(param.getValue(2));
