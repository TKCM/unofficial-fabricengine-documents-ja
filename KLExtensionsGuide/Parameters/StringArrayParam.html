StringArrayParam (object)¶

バージョン 1.12.0 で追加.

The StringArrayParam (object) implements the Param (interface) interface for String array values.





StringArrayParam param = :ref:`StringArrayParam <stringarrayparam>`(‘names’);
param.resize(3);
param.setValue(0, ‘Fabric’);
param.setValue(1, ‘Splice’);
param.setValue(2, ‘KL’);
report(param.getValue(2));
