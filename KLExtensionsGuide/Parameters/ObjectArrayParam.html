ObjectArrayParam (object)¶

バージョン 1.12.0 で追加.

The ObjectArrayParam (object) implements the Param (interface) interface for Object array values.





ObjectArrayParam param = :ref:`ObjectArrayParam <objectarrayparam>`(‘shapes’);
param.resize(2);
param.setValue(0, :ref:`PolygonMesh <polygonmesh>`());
param.setValue(1, :ref:`Lines <lines>`());
report(param.getValue(1));
