CompoundParam (object)¶

バージョン 1.12.0 で追加.

The CompoundParam (object) implements the Param (interface) interface for a nestable data structure. Compound parameters can be used to define a tree of values.





CompoundParam param = CompoundParam flags = param.addParam(:ref:`CompoundParam <compoundparam>`(‘flags’));
flags.addParam(:ref:`BooleanParam <booleanparam>`(‘visible’, true));
flags.addParam(:ref:`BooleanParam <booleanparam>`(‘transparent’, false));
flags.addParam(:ref:`BooleanParam <booleanparam>`(‘culling’, true));
