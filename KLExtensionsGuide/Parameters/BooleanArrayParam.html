BooleanArrayParam (object)¶

バージョン 1.12.0 で追加.

The BooleanArrayParam (object) implements the Param (interface) interface for Boolean array values.





BooleanArrayParam param = :ref:`BooleanArrayParam <booleanarrayparam>`(‘flag’);
param.resize(3);
param.setValue(0, true);
param.setValue(1, false);
param.setValue(2, true);
report(param.getValue(2));
