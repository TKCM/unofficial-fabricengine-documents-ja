ParamDict (object)¶

バージョン 1.12.0 で追加.

The ParamDict (object) object is used to store a list of parameters based on names.





ParamDict dict();


// store a scalar and a color
dict.setFloat64(‘length’, 1.7);
dict.setColor(‘diffuse’, :ref:`Color <color>`(1.0, 0.0, 0.0));


// restore them
report(dict.getFloat64(‘length’));
report(dict.getColor(‘diffuse’));
