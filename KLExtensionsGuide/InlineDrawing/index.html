InlineDrawing Extension¶

Fabric Engine version 2.0.0
Copyright 2010-2015 Fabric Software Inc. All rights reserved.

The InlineDrawing Extension provides a simple realtime rendering framework for direct rendering within DCCs such as Autodesk Maya, for example. The InlineDrawing Extension can also be used to render within the Fabric Engine standalone.

With the InlineDrawing Extension you can create renderable content in Fabric Engine, and a centralized render callback will then perform the rendering of the InlineDrawing‘s content. Please have a look at the OGLInlineDrawing for a code example.

For further information please refer to the InlineDrawing Guide.
