Containers Extension¶

Fabric Engine version 2.0.0
Copyright 2010-2015 Fabric Software Inc. All rights reserved.

The Containers Extension provides a series of low level data containers for managing bitmasks, memory and high performant index sets as well as localized arrays for a number of different data types.
