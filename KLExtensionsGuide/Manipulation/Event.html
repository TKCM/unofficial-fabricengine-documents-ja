Event (interface)¶

バージョン 1.12.0 で追加.

The Event interface for all supported event types.

参考

KeyEvent, MouseEvent, MouseWheelEvent

Functions in detail¶

Event.accept! ()

Sets the accept flag of the event object, the equivalent of calling setAccepted(true).

EventType Event.eventType? ()

Returns the event type id. The list of supported Ids are defined in ManipulationConstants.kl.

Event.ignore! ()

Clears the accept flag parameter of the event object, the equivalent of calling setAccepted(false).

Boolean Event.isAccepted? ()

Returns true if the event has been accepted by the active manipulator. The host application should then not propagate the event to other tools.

Event.setAccepted! ( in Boolean accepted )

when a manipulator processes and event, it should call either ‘accept’, or ‘setAccepted(true)’ to indicate that it has processed the event.
