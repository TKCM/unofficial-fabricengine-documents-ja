BaseEvent (object)¶

バージョン 1.12.0 で追加.

The BaseEvent object is the base class for all Event object types.

参考

EventDispatcher, Manipulator, Event, KeyEvent, MouseEvent, MouseWheelEvent

Members¶

Host host The Host object providing a method to query data about the host, and propagate data back to the host.
Viewport viewport The viewport that was in focus when the event was generated.
EventType eventType The id specifying the type of event.
Boolean accepted A boolean flag indicating that the event has been accepted, and no should not be handled by other manipulators or the host DCC application.
ModiferKey modifiers The keyboard modifier keys that were held when the event occurred.
