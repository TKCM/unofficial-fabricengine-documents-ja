Animation Extension¶

Fabric Engine version 2.0.0
Copyright 2010-2015 Fabric Software Inc. All rights reserved.

The Animation extension provides classes for defining keyframes, keyframe tracks, track sets, and clips.

The Keyframe type evaluation uses the same piecewize bezier interpolation found in the industry standard DCCs such as Maya, Softimage, and 3dsmax. In Maya, when a KeyframeTrack is added and connected to a Curve type, the KL Track type evaluates the the same results at the connected Maya FCurve node.

The Animation classes are used by the Characters Extension to drive the animation of characters. The FbxHelpers Extension generates a Clip data type when loading animation data from an Fbx file.
