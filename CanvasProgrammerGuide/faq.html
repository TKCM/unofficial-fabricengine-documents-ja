Canvas ではこれどうやるの？¶

ノードの実行のシリアライズは？

Canvas 中では明示的に指定しない場合、ノード実行順の保証はありません。Fabricバージョン 2.0.x では、Canvas 直列化機能を介しパイプすることで、ノードの実行順を次から次へと保証することができます。例：

dfgEntry {
  port1Out = port1In;
  port2Out = port2In;
}

このような関数では、そのポートを定められた順に “pull” することが保証され、実行順のシリアライズを通じ制御が可能になります。

注釈

このようなシリアライズの強制は、入力された値が並列化された事前処理が必要であると判明している場合、将来のバージョンにおいて変更される可能性があります。
